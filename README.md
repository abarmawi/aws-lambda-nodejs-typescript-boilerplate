# Rerento Backend API

Rerento aws lambda backend API

## Requirement

-   `nodejs 12`

## Features

-   `typescript` configured
-   `eslint` configured
-   `prettier` configured
-   `husky` pre commit configured

## Script

-   `npm run build`: build typescript
-   `npm run lint`: build code
-   `npm run prettifying`: run prettier to prettifying src files
